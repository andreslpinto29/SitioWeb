CSS Height and Width Las propiedades heighty widthse utilizan para establecer la altura y el ancho de un elemento. Ejemplo: div { height: 200px; width: 50%; background-color: powderblue;}
CSS Links Los cuatro estados de enlaces son: a:link - un enlace normal no visitado //a:visited - un enlace que el usuario ha visitado //a:hover - un enlace cuando el usuario pasa el mouse sobre él //a:active - un enlace en el momento en que se hace clic en él
Lists listas ordenadas(ol) y desordenadas(ul) Ejemplos ul.a { list-style-type: circle; } ul.b { list-style-type: square; } ol.c { list-style-type: upper-roman; } ol.d { list-style-type: lower-alpha; }
Tables (tr)//(td)
position Hay cinco valores de posición diferentes: static relative fixed absolute sticky. Ejemplo: div.static { position: static; border: 3px solid #73AD21; }//Ejemplo: div.relative { position: relative; left: 30px; border: 3px solid #73AD21; }//ejemplo fixed/fija: div.fixed { position: fixed; bottom: 0; right: 0; width: 300px; border: 3px solid #73AD21; }etc.
overflow La overflowpropiedad especifica si recortar el contenido o agregar barras de desplazamiento cuando el contenido de un elemento es demasiado grande para caber en el área especificada. La overflowpropiedad tiene los siguientes valores: visible- Defecto. El desbordamiento no está recortado. El contenido se renderiza fuera de la caja del elemento. hidden - El desbordamiento se recorta y el resto del contenido será invisible scroll - Se recorta el desbordamiento y se agrega una barra de desplazamiento para ver el resto del contenido auto- Similar a scroll, pero agrega barras de desplazamiento solo cuando es necesario
float La floatpropiedad se utiliza para posicionar y dar formato al contenido, por ejemplo, dejar que una imagen flote a la izquierda del texto en un contenedor. La floatpropiedad puede tener uno de los siguientes valores: left: el elemento flota a la izquierda de su contenedor right: el elemento flota a la derecha de su contenedor none: el elemento no flota (se mostrará justo donde aparece en el texto). Esto es predeterminado heredar: el elemento hereda el valor flotante de su padre EJEMPLO: img { float: right; }
